Калькулятор

Описание: Приложение используется для выполнения простых математических операций, таких как сложение, вычитание, умножение и деление. Графический интерфейс, реализованный с помощью PyQt5, предоставляет интуитивное управление для пользователей.

Необходимое ПО: Python3 ссылка: https://www.python.org/downloads/
Необходимые библиотеки: PyQt5 (установка через cmd: pip install PyQt5)

Алгоритм работы: 
1. cmd: python "путь до  файла\Calculator.py"
2. Выделите окно ввода и вводите цифры с помощью кнопок с обозначениями цифр
3. Выберите операцию (сложение, вычитание, умножение или деление) с помощью кнопок.
4. Нажмите кнопку "=" для получения результата.
